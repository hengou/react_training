### Scafolding d'une appli avec cli
npx create-react-app app --template=typescript

### type de composant
- UI
- Fonctional
- Layout

### structure de projet
- componant principal (même nom que le nom du composant principal. e.g: App)
- subdirectory "components/"

### Scope
